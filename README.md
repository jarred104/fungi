# Ineffable Fungi
## [ineffablefungi.com](http://ineffablefungi.com/)

This is the repository for my website, [ineffablefungi.com](http://ineffablefungi.com/).  The website is currently setup to have the [map](http://ineffablefungi.com/map.html) and [gallery](http://ineffablefungi.com/gallery.html) pages read a geoJSON file and generate content accordingly.

The [map](http://ineffablefungi.com/map.html) page uses the Leaflet and D3 JavaScript libraries to generate custom icons, popups, and cluster groups on a map using the information from the data source:

![map](https://github.com/jarred104/fungi/blob/master/static/images_readme/map.JPG)

--------------------------------------------------------------------------------------------------------------------------------

The [gallery](http://ineffablefungi.com/gallery.html) page uses the jQuery JavaScript library and Bootstrap to create a gallery image card for each item on the data source:
